Stock Market Analysis Project (R)
What this project does
This project looks at how different ETFs performed over time. It calculates returns, risk, and shows how $10,000 would grow if invested in each ETF.
ETFs Used:VOO(S&P500 ETF),VTI(Total Stock Market ETF), QQQM(NASDAQ-100 ETF), VT(Total World Stock Index Fund ETF), VUG(Growth Index Fund ETF), VXUS(Total International Stock Index Fund ETF)
What I Calculated: Daily returns, Growth of $10,000 over time, Risk (Volatility), Risk-adjusted return(Sharpe Ratio), Comparison to the S&P 500 (VOO)
Main Result: The project shows how different ETFs grow differently depending on risk and market exposure.
Tools Used: R(ggplot2,dplyr), Excel(data import)
